# Meridian Labs

Official website for [Meridian Labs](https://meridianlabs.us) — developer of exam preparation applications across beauty, healthcare, professional, and technology certifications.

## Overview

This repository contains the source for [meridianlabs.us](https://meridianlabs.us), which serves as the public-facing site for all Meridian Labs products. It includes:

- **Methodology** — Our adaptive learning approach and the science behind it
- **Privacy Policy** — Data handling practices across all applications
- **Terms of Service** — Usage terms and pass guarantee policy
- **Support** — Contact and help resources

## Deployment

Hosted on Cloudflare Pages with automatic deployment from the `main` branch.
